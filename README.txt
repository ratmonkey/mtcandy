Candy
by Kleopatra Angelidi
~*~*~*~*~*~*~*~
This is a mod for Minetest that will add fluff, color and diabetes in players life. It will also give the player a chance to make some nice candy houses, sweet shops or just use the blocks as decoration.
~*~*~*~*~*~*~*~
Rename the folder to candy.
